# ASTRA Frontend

## The project uses
• React 19.2.0
• Next.js 16.0.2
• Node.js 20.15.0
• Yarn

## How to run the project

1.	Make sure you are in the frontend directory:

`cd frontend`

2.	Ensure you are using Node.js 20.15.0 If you use nvm:

`nvm use`

3.	Install dependencies:

`yarn install`

(If you prefer npm: `npm install`)

4.	Start the development server:

`yarn dev`

(or `npm run dev`)

5.	Open the application:

`http://localhost:3000`

The system is currently configured to use http://localhost:3000 as the client base URL. The backend expects this exact URL, so please do not change it when running locally.
